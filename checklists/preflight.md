# Preflight Checklist

Use before non-trivial agent execution.

- [ ] Objective is explicit.
- [ ] Non-goals are listed.
- [ ] Prohibited substitutions are listed when near-miss outputs are plausible.
- [ ] Allowed files, tools, services, and external actions are clear.
- [ ] Data classification and secret/PII handling are clear.
- [ ] Risk tier is assigned.
- [ ] Stop conditions are listed.
- [ ] Validation method is known before execution.
- [ ] Executor and validator are separated when the task is non-trivial.
